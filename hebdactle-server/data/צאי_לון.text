צָאי לוּן (סינית: 蔡伦;‏ 50–121) היה סריס סיני, נחשב לממציא הנייר כפי שהוא מוכר כיום (בניגוד לפפירוס הקדום יותר אשר הומצא במצרים אלפי שנים קודם לכן).


== ביוגרפיה ==
צאי לון נולד בגְווֵייָאנְג, בירת מחוז גוויג'ואו בתקופת שושלת האן. שירת בחצר הקיסר כפקיד, ולצורך עבודות הרישום ניסה שיטות שונות להכנת נייר, עד להגעה לתהליך, אשר פרטיו אבדו עם הזמן, אשר איפשר הכנת נייר. בתמורה לעבודתו זו העניק לו הקיסר תואר אצולה וממון רב.
