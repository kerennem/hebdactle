אוסמיום (Osmium) הוא יסוד כימי מסדרת מתכות המעבר שסמלו הכימי Os ומספרו האטומי 76.


== תכונות ==
אוסמיום הוא היסוד המתכתי בעל הצפיפות הגבוהה ביותר מבין כל היסודות, 22.59 גרם/סמ"ק, מעט יותר מאירידיום, שצפיפותו 22.56 גרם/סמ"ק.
לאוסמיום יש את נקודת ההתכה הגבוהה ביותר (3033 מעלות צלזיוס) מיסודות קבוצת הפלטינה.


== שימושים ==
נוכח הרעילות הגבוהה של תחמוצותיו, כמעט ולא משתמשים באוסמיום בצורתו המתכתית.

אוסמיום טטרוקסיד (OsO4) שימש בעבר לזיהוי טביעות אצבעות.
שימוש ישן שנותר עד היום הוא "צביעה" בצילומים במיקרוסקופ אלקטרונים.
לסגסוגת של 90% פלטינה ו-10% אוסמיום שימוש בתור שתל בניתוחים.
אוסמיום טטרוקסיד משמש בתור מחמצן חשוב בריאקציות כימיות.
אוסמיום מהווה חלק מסגסוגת הנמצאת בחוד של מספר עטים נובעים, עקב קשיותו הגבוהה.


== היסטוריה ==
האוסמיום (מיוונית, אוסמה, שפירושו ריח) זוהה ב-1803 על ידי סמית'סון טננט בלונדון שבאנגליה יחד עם אירידיום במשקע של פלטינה מומסת במי מלך.


== צורה בטבע ==
אוסמיום מצוי בטבע בדרך כלל כסגסוגת עם אירידיום, פלטינה וניקל. מחצבים חשובים של אוסמיום נמצאים בדרום וצפון אמריקה.


== רעילות ==
אוסמיום טטרוקסיד הוא חומר רעיל במיוחד, שהרמה המקסימלית המותרת לחשיפה אליו (לשמונה שעות) היא 2 מיקרו-גרם לליטר אוויר. בריכוזים גבוהים יותר הוא יכול לגרום לגודש בריאות, לפגיעה בעור ובעיניים ולמוות.
