דה נאנג (בוייטנאמית: Đà Nẵng) היא העיר החמישית בגודלה בווייטנאם (אחרי הו צ'י מין סיטי, האנוי, הייפונג וקאנטחו), ואחת מערי הנמל החשובות בה. דה נאנג שוכנת לצידי ים סין הדרומי, ומהווה את העיר המרכזית במרכז וייטנאם. העיר ממוקמת בקרבת מספר אתרי מורשת עולמית, כגון העיר העתיקה הוי אן, ומקדש מי סון. בעבר הייתה שייכת העיר לדרום וייטנאם, ושכן בה בסיס אחד מבסיסי חיל האוויר הגדולים בווייטנאם, אשר שימש הן את הדרום וייטנאמים והן את האמריקאים בזמן מלחמת וייטנאם. הבסיס היווה נמל תעופה אשר בוצעו בו 2,595 המראות ונחיתות ביום בממוצע, עובדה שהפכה אותו לשדה התעופה העמוס ביותר בעולם באותה התקופה. הבסיס הפך לאחר המלחמה לנמל התעופה הבינלאומי דה נאנג.
