== דמוגרפיה ==
בשנת 2006 כלל אזור מערב ארצות הברית, 69,355,643 תושבים. אזור מערב ארצות הברית כולל הן את המדינה הכי מאוכלסת בארצות הברית (קליפורניה) והן את המדינה הכי פחות מאוכלסת בארצות הברית (וויומינג).
במדינות מערב ארצות הברית מספר רב של מיעוטים אתניים. שלוש מארבע המדינות בהן האוכלוסייה הלבנה אינה מהווה רוב נמצאים באזור זה - הוואי, קליפורניה וניו מקסיקו.
