חמים וטעים (באנגלית: Some Like It Hot) הוא סרט קולנוע קומי אמריקאי משנת 1959, בכיכובם של מרילין מונרו, טוני קרטיס וג'ק למון, בבימויו של בילי ויילדר וכותב הסיפור של הסרט אוטו ליסאואר בשם הספרותי מיכאל לוגן.
בשנת 2000 בחר מכון הסרטים האמריקאי סרט זה לסרט הקומי האמריקאי המצחיק ביותר בכל הזמנים. בשנת 2017, הוכתר הסרט כקומדיה הטובה בכל הזמנים, במסגרת סקר שערכה רשת בי-בי-סי בין 253 מבקרי קולנוע מ-52 מדינות.


== עלילה ==
העלילה מתרחשת בשנת 1929, ב"תקופת היובש" ובימים שקדמו לקריסת וול סטריט ולראשית השפל הגדול. שני נגנים, ג'ו וג'רי (קרטיס ולמון), נמלטים מהמאפיה, לאחר שהיו עדים במקרה למעשה רצח בשיקגו, המזכיר את טבח יום ולנטיין. הם נמלטים מהעיר, מתחפשים לנשים (בשמות הבדויים ג'וזפין ודפני) ומצטרפים ללהקת נשים העושה את דרכה ברכבת להופעות במלון בפלורידה. שניהם מתאהבים בדרך בחברת הלהקה "שוגר" (מונרו). מעצם היות השניים מחופשים לנשים, נוצרות שפע סצינות משעשעות. בין היתר, פנסיונר מיליונר בשם אוסגוד מתאהב בג'רי (דפני), ג'ו (ג'וזפין) מתחפש בעצמו לאיל נפט קר רוח, במטרה לכבוש את ליבה של שוגר, ובהמשך מגיעים אנשי המאפיה (בראשות "בונפארט"- מנהיג המאפיה, המגולם בידי נחמיה פרסוף) לאותו מלון, ומזהים כי הנשים המחופשות הם העדים לרצח.
הסרט מסתיים כאשר ג'רי מנסה לשכנע את אוסגוד שלא יוכל להינשא לו, מסיר את הפאה וקורא: "אני גבר!". אוסגוד עונה במשפט הסיום המפורסם: "Well, nobody's perfect" ("ובכן, איש איננו מושלם").
