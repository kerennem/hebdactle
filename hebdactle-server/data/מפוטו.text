מפוטו (באנגלית: Maputo) היא עיר הבירה של מוזמביק. היא שוכנת בדרום המדינה, לחוף האוקיינוס ההודי, ומתגוררים בה כמיליון ורבע תושבים.


== כלכלה ==
כלכלת העיר מבוססת על הנמל הממוקם בעיר, ועל ייצוא של פחם, כותנה, סוכר, כרום, קופרה וסיסל. בעיר מייצרים מלט, כלי חרס, רהיטים, נעליים וגומי.


== מיקום העיר ==
העיר ממוקמת בצידו המערבי של מפרץ מפוטו, בפתח נהר טמבה.


== היסטוריה ==
העיר נוסדה במאה ה-18 בשם לורנסו מארקש על שם הסוחר הפורטוגזי לורנסו מארקש, אשר חקר את האזור בשנת 1544.
בשנת 1895, עם הקמת מסילת הרכבת לפרטוריה בדרום אפריקה החלה העיר להתפתח. העיר תוכננה כעיר קולוניאלית טיפוסית, עם שדרות רחבות ורחובות מתוכננים. בשנת 1898 הפכה לבירת מוזמביק. כל עוד הייתה מוזמביק קולוניה פורטוגזית, ישבה בעיר קהילה פורטוגזית גדולה ומבוססת. לאחר שמוזמביק קיבלה עצמאות מפורטוגל בשנת 1975, הוחלף שמה של העיר למפוטו, ותושביה הפורטוגזים נמלטו ממנה בהמוניהם.


== השכלה ==
בעיר נמצאת אוניברסיטת אדוארדו מונדליין.


== אתרים בעיר ==
בעיר נמצאים המוזיאון להיסטוריה של מוזמביק, המוזיאון הצבאי, וכן הקתדרלה הקתולית גבירתנו של פטימה.
