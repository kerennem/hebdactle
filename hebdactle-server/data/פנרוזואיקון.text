פנרוזואיקון (מיוונית עתיקה: φανερός (פנרוס - נראה לעין) ו-ζῷον (זואון - בעל חיים)) הוא עידן על המציין את הפרק המאוחר בלוח הזמנים הגאולוגי של כדור הארץ. 
משך עידן-העל כחצי מיליארד שנים, החל מהופעת בעלי חיים נראים לעין ועד ימינו. 
הפנרוזואיקון מתחלק לשלושה עידנים: 


== ראו גם ==
לוח הזמנים הגאולוגי
יחידת זמן גאולוגית


== קישורים חיצוניים ==
פנרוזואיקון, באתר אנציקלופדיה בריטניקה (באנגלית)
