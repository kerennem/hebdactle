הומיניניים (שם מדעי: Hominini), שבט בתת-משפחת הומיננאים, שהתפתח מהמין אורורין טוגננסיס (Orrorin tugenensis) וכולל את:

תת-שבט פאנינים (Paninina) בעל סוג יחיד שימפנזה (Pan)
תת-שבט הומינינים (Hominina)כל ההומיניניים חסרי בלוטה במצח (זאת בניגוד לשבט הגורילניים).


== קישורים חיצוניים ==

הומיניניים, באתר אנציקלופדיה בריטניקה (באנגלית)
