אונדטרה (שם מדעי: Ondatra zibethicus) היא מין מכרסמים ממשפחת האוגריים, בו סוג ומין יחיד. היא מכונה גם חולדת המושק עקב ריח המושק המאפיין את פרוותה.


== מאפיינים ==
תפוצתה של האונדטרה באמריקה הצפונית, בעיקר בקנדה, ארצות הברית וצפון מקסיקו. במאה ה-20 הוכנסה בידי האדם גם לאירופה הצפונית-מערבית, שם היא נחשבת למין פולש. האונדטרה לא הוכנסה לישראל, אליה הוכנס מכרסם שוכן מים-מתוקים ממשפחה אחרת, הנוטריה, שמוצאה באמריקה הדרומית.
האונדטרה יכולה להגיע לאורך 40–70 ס"מ ולמשקל 0.6-2 ק"ג. היא דומה במראה לאוגר או לבונה ובעלת פרווה הדוחה מים וזנב שרירי המשמש כהגה במים. היא ניזונה בעיקר מצמחי מים המהווים כ-95% מתזונתה ואולם היא תגוון בחרקי מים, צדפות, סרטני נהרות, צפרדעים, ראשנים, עופות וצבים קטנים.
האונטרות חיות בקבוצות קטנות סביב גדות האגמים והבריכות. באמריקה נצפו אונדטרות ששיתפו פעולה עם בונים בבניית הסכר וניזונו מהגזעים שלו בתוכו.
