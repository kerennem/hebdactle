פיגום הוא מבנה ארעי, המשמש לתמיכת בני אדם וציוד כדי לבנות או לשפץ מבנים גדולים יותר. פיגומים הם, בדרך כלל, מערכת מודולרית של צינורות פלדה, אך ניתן להשתמש גם בחומרים אחרים, כגון במבוק, כנהוג במספר מדינות באסיה, כגון הונג קונג.
התקנת פיגומים ושימוש בהם טומנים בחובם סכנות בטיחות פוטנציאליות.


== פיגומים בישראל ==
הפיגומים בהם נעשה שימוש בישראל מקורם בעיקר במפעלים באזור ג'נין[דרוש מקור] ואיכותם נחשבת לנמוכה בעיקר בתחום הבטיחותי. כתוצאה מרשלנות אך גם מחוסר יציבותם של הפיגומים, כמו גם איכות פרטיהם, נפגעו פועלי בניין רבים במהלך השנים כתוצאה מפיגומים שהתפרקו או איבדו את יציבותם. כתוצאה מכך, הוחל ב-2018 להשתמש בפיגומים עם תקן אירופי. כמו כן, בעקבות העלייה במספר ההורגים והפצועים באתרי בנייה והודות לפעילות ציבורית ותקשורתית שהדגישה את הנושא, נקבעו תקנות בחוק המחייבות את כל אתרי הבנייה באופן הדרגתי להשתמש בפיגומים עם תקינה בלבד.
