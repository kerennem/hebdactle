יקינתון (שם מדעי: Hyacinthus; שמות עבריים: כדכוד, יקנת) הוא סוג צמח במשפחת היקינתוניים.


== אטימולוגיה ==
מקור השם הוא במיתולוגיה היוונית. יקינתון (יאקינתוס) היה שמו של נער יפה תואר, בנו של פיראוס מלך מקדוניה, שאפולו, האל האולימפי, התאהב בו. משהתחרה אפולו עם יקינתון בזריקת דיסקוס, קינא בו אל רוח המערב זפירוס ולכן גרם לכך שהדיסקוס שהשליך אפולו יפגע בראשו של יקינתון. הנער מת, ומדמו עלה צמח הנושא את שמו.


== טקסונומיה ==
הסוג כולל שלושה מינים שתפוצתם באזור הים תיכוני ובאפריקה.

Hyacinthus transcaspicus


== מאפיינים ==
היקינתון הוא גאופיט בעל בצל רב שנתי. עליו הצרים מרוכזים בבסיס הצמח, והפרחים דמויי פעמון או משפך וסדורים באשכול על עמוד תפרחת בלתי מסועף. צבעי הפרח משתנים ממין למין כאשר הגוונים הנפוצים הם כחול, תכלת, סגול, ורוד ולבן.
הצמח תורבת בהולנד בסוף המאה השש-עשרה ומשמש בעיקר לנוי. הוא טופח בשני כיוונים: לתפרחת צפופה יותר, רבת פרחים, ולזנים צבעוניים. זהו הפרח היחיד כמעט שלא השתנה בצורתו התרבותית מצורת הבר.


== בתרבות ==
חוקרים רבים נוטים לזהות את היקינתון עם השושנה הנזכרת בתנ"ך, בספר "שיר השירים".
לאה גולדברג כתבה את שיר הילדים "פזמון ליקינתון", אותו הלחינה רבקה גוילי, ולו ביצועים רבים.
