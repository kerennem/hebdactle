קֵטוֹנים (Ketones) הם קבוצה של תרכובות אורגניות, שכולן מכילות את הקבוצה הפונקציונלית קרבוניל (C=O, אטום פחמן הקשור בקשר כפול לאטום חמצן). בין הקטונים חומרים ידועים רבים, אשר הידוע מכולם הוא הממס האורגני אצטון, והוא גם הקטון בעל המבנה הפשוט ביותר, והוא אשר הקנה לקבוצה את שמה: השם "קטון" הוא שיבוש של "אצטון", שמקורו מהמילה הלטינית: אצטוּם, שפירושה "חומץ". קטונים רבים נותנים לפירות את טעמם וריחם.
בתרכובות שהן קטונים ואלדהידים כאחד (בהן קיימת קבוצת קרבוניל לצד קבוצת אלדהיד), מקבלת קבוצת הקרבוניל את השם אוֹקסוֹ.
על מתן שמות לקטונים, ראו מונחון IUPAC. בקצרה: שמו של קטון מורכב מתחילית המציינת את מספר אטומי הפחמן במולקולה, מהסופית -אָנוֹן, ומספרה המציינת את מיקומה של קבוצת הקרבוניל במולקולה.
הקטון הפשוט (והידוע) ביותר הוא פּרוֹפּנוֹן (אצטון, Propanone, Acetone), המכיל שלושה אטומי פחמן. 
אצטואצטט ואצטון נוצרים בכבד בתהליך פירוק חומצות שומן ונקראים גם גופי קטו.


== הפקת קטונים ==
