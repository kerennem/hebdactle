סיסיים (שם מדעי: Apodidae) היא משפחת עופות מסדרת הסיסאים, והיא המשפחה היחידה בסדרה זו שמיניה נפוצים באירופה ובישראל.


== מאפיינים ==
הסיסיים הם עופות דקי גזרה ובעלי כנפיים ארוכות ומחודדות. הם מעופפים מיומנים ביותר ומבלים את מרבית חייהם בתעופה - כך הם ניזונים מחרקים מעופפים, ישנים ואף מזדווגים. רגליהם אינם מותאמות להליכה או לעמידה על ענפים, אך מותאמות לאחיזה בקירות. הם מקננים בסדקי מבנים ובמצוקים.
יש הרואים את הסיסיים כמקורבים לסנוניתיים בשל מראם ומעופם המהיר, אולם מדובר באבולוציה מתכנסת. הסיסיים מקורבים גנטית לקוליבריים. ניתן לראות דמיון במבנה הראש והגוף, בכנפיים וביכולות התעופה המרשימות.
