אַמַנִיטָה מוּסְכַּרִיָה (וכן אמנית הזבובים ואמנית זבובנית; שם מדעי: Amanita muscaria) היא פטריה מהסוג אמנית. פטריה זו היא רעילה ובעלת השפעה פסיכואקטיבית. מזוהה בקלות באמצעות המראה הבולט שלה: כובע אדום עם נקודות לבנות.


== רעילות ==
אמניטה מוסכריה מכילה חומצה איבוטונית אשר אכילתה מסוכנת לאדם ויכולה להוביל למותו. תכונתה כפטריית הזיות הפכה אותה לחלק מהפולחן השמאני, שבו הנפש כביכול יוצאת מהגוף ומגיעה למישורי תודעה אחרים. במסע אחר סיפר דנדן בולוטין על התנסותו בפטריות הזיה אצל ידידו השמאני, ואף הזהיר שתרמילאים ישראלים אחרים שהתנסו מתו או אושפזו בבתי חולים פסיכיאטריים.
על אף סיווגה כמעוררת הזיות, בישראל אמניטה מוסכריה אינה כלולה בפקודת הסמים המסוכנים. זאת בניגוד לפטריות פסילוסיבין ("פטריות קסם") אשר מיצויי החומר הפעיל בהן נחשב בלתי חוקי (אחזקת הפטריה אינה נחשבת לעבירה פלילית שכן הן אינן נכללות בפקודת הסמים המסוכנים).


== אמניטה מוסכריה בתרבות ==
בשל מראה, הפטרייה מופיעה ביצירות הפונות לילדים. בספרי ילדים ובציורי קיר בגני ילדים נפוץ הדימוי של ננס, בעל פני תינוק או מזוקן, המתגורר בפטריית האמניטה הצבעונית. פטריות חומר וחרסינה, עם או בלי צביעה באדום ולבן, מלוות פעמים רבות גמדי גינה בגינות ובחצרות. 
אמניטה מוסכריה בצבעים שונים נהייתה דימוי המזכיר את אליס בארץ הפלאות, שם מסתובבת אליס בעולם החלום וביער של פטריות. עוד קטעים הקשורים לדימוי הסם: אליס שותה שיקוי שמשפיע עליה והיא נהיית אצבעונית; הזחל העצלן שמעשן נרגילה ומפזר את עשנו באוויר. בסדרת המשחקים Super Mario Bros, אמניטה מוסכריה היא הפטריה שמריו אוכל כדי לגדול.
